My Projects

Game Discovery

My fourth project was built using Javascript, React, Node, Express, MongoDB, Mongoose, and the IGDB API. It is a game search engine designed for lost gamers looking to discover their next game to play. This project was my first full-stack React app and took place during my twelfth week of WDI and was done in four days.

Don't Think, Just Eat

My third project was built using Javascript, Node, Express, MongoDB, Mongoose, and the Yelp API. You are able to put in a desired location and find a random restaurant within one mile. This project was a group project and was done alongside three others. We took the mob programming approach with one driver and three navigators. My designated role was Scrum Master, maintaining an organized workflow throughout the project.

Deployed App | GitHub Repo

WAR

My first project was using JavaScript, CSS, and HTML to create the card game, War, where two players simultaneously play a card, and the player with the higher value card takes both cards into their deck. This project took place during my third week of WDI and was done in four days.

Deployed App | GitHub Repo

GL HF

The second project was built using Ruby on Rails to create a CRUD application revolving around reviews for video games. You are able to create a profile and submit reviews for specific games and rate them on a scale of 100. This project took place during my sixth week of WDI and was done in three days.
